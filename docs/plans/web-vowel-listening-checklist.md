# Web 母音プロトタイプ試聴チェックリスト

## 概要

この文書は、Web 側の初期母音プロトタイプを試聴する際に、最低限どこを確認するかを整理した短いチェックリストである。

現段階では、高忠実度な評価ではなく、母音差が分かるか、操作に対して不自然な挙動がないかを確認することを目的とする。

## 確認項目

### 1. 母音差が聞き分けられるか

- `/a/`, `/i/`, `/u/`, `/e/`, `/o/` を切り替えたときに、音色差が明確にあるか
- どの母音も、ほぼ同じ音に聞こえていないか
- 少なくとも `/a/` と `/i/` の差がはっきり分かるか

### 2. 母音の印象が極端に不自然ではないか

- 特定の母音だけが極端に耳障りになっていないか
- 金属的すぎる、笛のようすぎる、単なるフィルタ付き発振器にしか聞こえない状態が強すぎないか
- 母音らしさの方向に最低限寄っているか

### 3. `Pitch` の変更が不自然でないか

- ピッチを上げ下げしたときに、音量や音色が極端に破綻しないか
- ピッチ変更によって母音差が急に消えないか

### 4. `Tract Scale` の効き方が粗くても納得できるか

- `Tract Scale` を変えたときに、フォルマント全体が動いている印象があるか
- 少し動かしただけで音が壊れたり、急に別物になりすぎないか
- 値を上下したときに、声道長の変化らしい方向性を感じられるか

### 5. `Gain` の変更が素直か

- 音量調整として自然に効くか
- ゲイン変更で意図せず歪みや極端な変色が起きていないか

### 6. UI 操作と音の関係が分かりやすいか

- 母音切り替え、ピッチ、声道スケールの役割が直感的に分かるか
- スライダーを動かした結果が音として追いやすいか

## 試聴時にメモしておくとよいこと

- どの母音差が分かりやすく、どの組み合わせが分かりにくいか
- `Tract Scale` のどの範囲で不自然さが強いか
- もっと効きを強くしたいパラメータ、弱くしたいパラメータは何か
- 次に追加したい要素が、フォルマント調整なのか、励振源改善なのか、UI 改善なのか

## 備考

このチェックリストは、初期段階の試聴観点を揃えるための簡易メモであり、実装の進展に応じて更新する前提の草稿である。
