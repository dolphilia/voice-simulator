# Project Overview

## プロジェクト概要

このリポジトリは、声道、声帯、喉頭などの発声機構をソフトウェア上で物理的にシミュレーションし、母音や子音を含む音声の生成を目指すプロジェクトである。

音声合成には、録音音声の断片を接続する方式や、深層学習を用いる方式など、さまざまな手法が存在する。しかし本プロジェクトが主に扱うのは、それらとは異なり、人間の発声器官そのものの構造や振る舞いをモデル化し、そこから音声を生み出すアプローチである。

方向性として近い先行例に Pink Trombone がある。Pink Trombone は、口腔や舌、喉の形状変化によって声がどのように変わるかを、視覚的・聴覚的・構造的に理解できるシミュレーターである。本プロジェクトでは、これを重要な参考例としつつ、最終的にはそれ以上に高度なシミュレーターを目標とする。たとえば、声道の材質や形状、声帯そのものの調整、子音の生成、さらに踏み込んだ物理シミュレーションを含む音声生成などを視野に入れている。

- [Pink Trombone](https://dood.al/pinktrombone/)
- [Pink Trombone (Imaginary)](https://www.imaginary.org/program/pink-trombone)

また、表現方法についても検討対象とする。Pink Trombone は 2D ベースの表現を採用しているが、本プロジェクトでは 2D で十分なのか、あるいは 3D 表現まで拡張すべきかも含めて検討していく。あわせて、直感的に扱えることと、細かな制御が可能であることの両立も重視する。

## 位置づけ

本プロジェクトは、実世界の声道模型、人工喉頭、人工声帯といった研究・装置に近い発想を、ソフトウェア上で再現しようとする試みでもある。その意味で、音楽制作で用いられる物理モデリング音源に近い性質を持ち、人間の声を対象とした物理モデリング音源を作るプロジェクトとも言える。

一方で、録音音声の接続による音声合成や、深層学習を中心とした音声生成は、本プロジェクトの主目的ではない。ただし、それらを完全に排除するわけではなく、比較研究や評価、補助的な検討のために扱うことはあり得る。

## 研究・実装の対象

研究段階では、隠れマルコフモデルのような比較的古典的な手法を含め、スペクトル、ケプストラム、フォルマントの解析や合成なども扱う可能性がある。フーリエ変換は基礎的な要素として頻繁に登場するはずであり、試験的な音声生成や分析のために、イコライザーなどの音響処理ツールを実装することも考えられる。そのため、サウンドプログラミングの知識と技術は中核的な要素となる。

## 開発環境について

開発環境は現時点では未定であり、今後の重要な検討事項である。候補を選ぶ際には、少なくとも以下の条件を満たす必要がある。

- 音声を扱いやすいこと
- 物理シミュレーションや数値計算を行いやすいこと
- 視覚的な表現やインタラクティブな操作系を実装しやすいこと

つまり、音響処理、物理計算、可視化・UI の三要素を無理なく扱える技術スタックを選定する必要がある。

## 資料収集と調査方針

関連資料は継続的に収集していく。近年は深層学習ベースの音声生成が目立つ一方で、本プロジェクトに直接関係する物理的・構造的アプローチの研究も一定数存在する。参考になりそうな論文や研究成果を定期的に調査し、整理・蓄積していく。

調査対象は、音声生成そのものに限らない。音の物理学、空気振動、材質による音の変化、音響・反響シミュレーション、物理モデリング音源、サウンドエフェクト、さらにそれらを支える計算手法やプログラミング技術まで、関連しうる広い分野を対象とする。
