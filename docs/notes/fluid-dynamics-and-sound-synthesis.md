# 調査メモ: 流体力学と関連分野の音生成・音声合成への応用

作成日：2026-04-12

---

## 概要

本メモは、このプロジェクトで現在進行中の研究（DWM、波動管モデル、フォルマントフィルタ）とは独立した視点から、**流体力学およびその周辺分野が音の生成・音声合成にどう関与するか**を体系的に調査・考察したものである。

既存の `alternative-synthesis-methods.md` では「流体力学ベースの統合モデル」を一節として取り上げたが、本メモはその内容を大幅に掘り下げ、具体的な理論・手法・研究事例・応用可能性を整理することを目的とする。

## 1. 音源理論としての空気力学音響学（Aeroacoustics）

### 1.1 Lighthill の音響アナロジー（1952）

音の発生における流体力学的な統一理論の出発点は、James Lighthill が 1952 年に提唱した**音響アナロジー（Acoustic Analogy）**である。

Lighthill の発想は、Navier-Stokes 方程式を変形して

```
□²ρ' = ∂²T_ij / ∂x_i ∂x_j
```

という形式（波動演算子 □² に右辺の音源項を組み合わせた方程式）に書き換えることにある。右辺の **Lighthill テンソル T_ij** は流れの乱流応力・粘性応力・圧力変動などを含み、これを等価な「四重極（quadrupole）音源」と見なすことで、近くの乱流流れが遠方に音として伝播する仕組みを定量化できる。

この定式化の意義は以下の点にある。

- 流体力学と音響学を統一的に扱える理論的な骨格を提供した
- 音源を「単極（monopole）」「双極（dipole）」「四重極（quadrupole）」に分類する枠組みを与えた
- 音源ごとに速度依存性が異なること（単極 ∝ U⁴、双極 ∝ U⁶、四重極 ∝ U⁸）が明確になった
- 発声のような低マッハ数条件では双極・単極が支配的であることが判明した

その後 Curle（1955）が固体境界を含む系への拡張を行い、Powell（1964）が渦度（vorticity）を音源とする**渦音理論**を整備し、Howe（1975）が渦音の定式化をさらに洗練した。

### 1.2 Ffowcs Williams-Hawkings（FW-H）方程式（1969）

Lighthill アナロジーを**動く固体境界が存在する系**に拡張したのが FW-H 方程式である。音源は3成分に分解される。

| 音源種別 | 物理的起源 | 声道における対応 |
|---|---|---|
| 単極（Monopole） | 固体境界の体積変化 | 声門の開閉による体積流量変動 |
| 双極（Dipole） | 壁面に作用する非定常力 | 声帯・歯・口唇への気流衝突 |
| 四重極（Quadrupole） | 流体内部の乱流応力 | 声門後方の乱流ジェット |

低マッハ数条件（発声時の声道内流速は通常 0.01〜0.05 Mach 程度）では四重極の寄与は比較的小さく、単極と双極が支配的であることが Zhao ら（JASA, 2002）による圧縮性 Navier-Stokes シミュレーションで定量的に確認されている。

この分解は実用的に重要で、**音源種別ごとの近似モデルを組み合わせる設計**への道筋を与える。

### 1.3 渦音理論（Vortex Sound Theory）

Howe（2003, Cambridge University Press「Theory of Vortex Sound」）は、音響源を渦度の時間変動として定式化し、

```
音響パワー ∝ ∮ (ω × v) · ∇φ dV
```

という積分形で音源強度を表した。ここで ω は渦度、v は速度、φ は音響ポテンシャルである。この理論によれば、**渦が音響場の節（音圧最大点）付近に位置するとき音響変換効率が最大**になる。

発声との関連では、声帯後方に形成されたジェット渦が声道の特定共鳴モードと空間的に重なるとき、音響エネルギーの変換が効率的に起きると説明できる。

## 2. 声門・声帯の流体力学

### 2.1 Bernoulli 効果と Coanda 効果

声帯振動の駆動力について、かつては van den Berg（1950年代）による **Bernoulli 吸引**が主因とされていたが、現在の理解では以下のように整理されている。

- **Bernoulli 効果**は声門内に実在するが、声帯閉鎖の主因ではない
- 声帯を閉じる主要な力は、声門チャネルの「収束–発散」幾何形状が作り出す**非対称圧力分布**である
- 声門後部では流れが剥離し、声帯後方に強い低圧域が生じることで粘膜が引き戻される

一方、**Coanda 効果（壁面付着効果）**は声門後方のジェット偏向を引き起こす。

- 声門から放出されたジェットは、声道壁の片側に引き寄せられる傾向がある
- Neubauer ら（JASA, 2007）の静的声帯模型実験では、声門後方のジェットが左右どちらかの壁に付着することが高速PIV計測で確認されている
- この**非対称ジェット偏向**は声帯の非対称振動（左右差）と相互に影響し合う

Coanda 効果は発声のモデル化において見落とされがちだが、特に**ブレシー（breathy）な発声や不規則発声（jitter/shimmer）**の原因の一つとして重要である。

### 2.2 声門後方のジェット流と渦形成

声門を通過した気流は以下の段階を経て乱流に遷移する。

```
声門狭窄部 → ジェット形成 → せん断層不安定 → 渦巻き込み → 渦放出 → 乱流遷移
```

声門後方の渦ダイナミクスに関する研究（Drechsel & Thomson, JASA 2008; PMC記録 Kamenev ら 2021）では、以下の特性が報告されている。

- 声門からのジェットは周期的な渦放出（Kelvin-Helmholtz 不安定）を示す
- 偽声帯（false vocal folds）が存在する場合、これが渦の発達を干渉し音色を変える
- 渦形成時間（vortex formation time）は声門開放時間と密接に関係し、発声スタイルの違いが渦構造の違いとして現れる

音声合成の観点では、この渦形成過程の違いが**声質（voice quality）の差**に対応する。すなわち、圧迫声（pressed voice）・常態発声・ブレシー声・気息音の違いは、声門開放比・ジェット速度・渦形成特性の違いとして流体力学的に説明できる。

### 2.3 流体–構造連成（FSI：Fluid-Structure Interaction）

声帯振動の完全な記述には、声帯組織（固体）と気流（流体）の連成が必要である。

FSI モデルの発展経緯：

| 世代 | アプローチ | 代表例 |
|---|---|---|
| 第1世代 | 集中定数 + 準定常1D流れ | 2質量モデル（Ishizaka & Flanagan, 1972） |
| 第2世代 | FEM固体 + 1D流れ | 連続体声帯 + Bernoulli流モデル |
| 第3世代 | FEM固体 + 3D圧縮性流体（LES/DNS） | Zheng ら（JASA 2011）、immersed boundary 法 |

Zheng, Mittal, Xue, Bielamowicz（JASA, 2011）による3次元 DNS+IB/FEM 連成は、現時点での最高精度な数値声帯モデルの一つである。収束–発散声門形状の再現、ジェットの3次元構造、声帯の非対称振動など、実験と良く一致する挙動が得られている。

### 2.4 声門ジオメトリーと圧力分布（Scherer の研究）

Ronald C. Scherer（Bowling Green State University）らは数十年にわたって声門内の圧力分布を実験・CFD で測定し、次の知見を積み重ねた。

- 声門入口の丸み（convergent inlet radius）が声帯への推進力に大きく影響する
- 声門出口の発散角度が流れ剥離点を決め、音圧レベルに直結する
- 声門形状の微小な非対称性が圧力分布の非対称性を生み、左右声帯の振動差につながる

これらの知見は、声帯モデルに「形状パラメータ」を導入する設計の根拠となる。

## 3. 声道内の流れと音源

### 3.1 有声音発声時の声道内流れ

通常の有声音発声では、声道内の気流は比較的遅く（平均流速 0.1〜1 m/s）、Reynolds 数は概ね 1,000〜5,000 程度に留まる。この条件では**層流支配**であり、音源は声帯の振動によって生成される体積速度パルスが主である。

声道は音響フィルタとして機能し、声帯が生成した波形を整形する。波動管モデルや DWM はこの「声道=音響フィルタ」機能を近似する手法である。

### 3.2 摩擦音生成時の乱流

/s/, /ʃ/, /f/, /θ/, /ç/ などの摩擦音（fricatives）では、状況が大きく異なる。

- 声道収縮部（歯列間・口蓋–舌間など）での流速が大幅に上昇する
- 局所 Reynolds 数が 2,000〜10,000 に達し、乱流遷移が起きる
- 収縮後方に形成されたジェットが歯・口唇・口蓋などの障害物に衝突し、**双極音源**を生成する

Shadle（Proceedings of Royal Society A, 2004）の /s/ の空気力学音響学研究によれば、前歯直後の衝突点が主要な双極音源であり、音源強度は局所流速の約5乗に比例することが実験で確認されている。

摩擦音の音源は2種類に分類される（Narayanan & Alwan, IEEE 2000）。

1. **障害物音源（obstacle source）**：乱流ジェットが固体障害物（歯・口唇）に直撃して生じる双極源。/s/、/ʃ/ など歯擦音の主要音源。
2. **壁面音源（wall source）**：ジェットが壁面に斜めに衝突して生じる音源。/ç/（ドイツ語「ich」）、/x/ などで主体。

### 3.3 息成分（Aspiration）の音源

声帯が完全に閉じない有声音や、声帯閉鎖前後の過渡的な気流（onset/offset aspiration）では、声門付近での乱流が主要な音源となる。

- 声門通過時の Reynolds 数が閾値を超えると乱流が発生
- この乱流ノイズが声道を通じて整形され、**ブレシーな声質**の成分となる
- 閾値は声門開度（glottal area）と気流速度の積（体積速度 U）で決まり、おおよそ U > 200〜400 cm³/s 付近で乱流遷移が起きる

### 3.4 鼻腔内流れ

鼻腔は独特の形状的特性を持つ。

- 鼻甲介（turbinates）が複雑な渦流を生成する
- 通常の発声音域では層流的流れが支配的
- 軟口蓋（velum）の開閉が鼻腔への気流配分を制御し、鼻音・鼻母音を生じる
- 鼻咽腔閉鎖不全（VPI）がある場合、鼻腔内で乱流遷移が起き、異常な鼻摩擦音が生じる（CFD による定量分析あり：PMC6542651）

## 4. 数値手法の比較と音声合成への適用性

### 4.1 LES（Large Eddy Simulation）

現時点で声道・喉頭研究に最も広く使われている高精度手法。

**原理**：大スケールの渦構造を直接解像し、サブグリッドスケールのみモデル化する。乱流の大部分のエネルギーを担う大スケール渦が形状・境界条件に強く依存するため、直接解像が重要。

代表的研究：
- Schickhofer, Malinen, Mihaescu（JASA 2019）：MRI 取得の現実声道形状を用いた圧縮性 LES
- Mihaescu ら（KTH）：ANSYS Fluent を用いた声帯後方の声道内流れ解析

**計算コスト**：10振動サイクルのシミュレーションに最適化後でも約70時間（CPU 並列、StarCCM+ 有限体積法）。音声合成への直接リアルタイム利用は不可能。

### 4.2 DNS（Direct Numerical Simulation）

乱流のすべてのスケールを直接解像する。

**原理**：Navier-Stokes 方程式をすべてのスケールで解く。音響まで含めた完全な連成を実現できるが、格子数 N の増大が Reynolds 数に対して Re^(9/4) で増えるため計算コストが急激に大きくなる。

代表的研究：
- Zheng, Mittal, Xue, Bielamowicz（JASA 2011, PMC3155594）：3次元喉頭モデルでの DNS + immersed boundary / FEM 連成

**計算コスト**：声帯振動を含む完全な発声シミュレーションを DNS で行うには、数千コア・数日規模の計算が必要。近未来においても音声合成のリアルタイム利用は非現実的。

### 4.3 格子ボルツマン法（LBM：Lattice Boltzmann Method）

Navier-Stokes の間接的解法で、分子的な衝突・移流過程を離散格子上で模擬する。

**音響との親和性**：LBM は弱圧縮性の性質を内包しており、音波の自然な伝播が可能。Navier-Stokes に基づく手法と異なり、音と流れを一つの枠組みで同時に扱える。

主な特徴：
- **複雑形状への高い適応性**：専用メッシュジェネレーターが不要（デカルト格子上で境界を表現）
- **並列化が容易**：局所的な計算のみで更新するため、大規模並列計算と相性が良い
- **弱圧縮性**：低マッハ数音響計算においても精度が保たれる
- **実装の相対的な単純さ**：Navier-Stokes ソルバーより実装が単純

発声への適用研究として Freund ら（JASA 2009, PMC2677327）が LBM の声門流への応用可能性を検討し、複雑形状への適応性の高さを確認している。

**計算コスト**：LES に比べれば有利だが、数十秒の音声を生成するには現状でも相当な計算量が必要。ただし、**声門周辺の局所モデルに限定**すれば、Python + NumPy レベルの実装で研究用の試作は可能。

### 4.4 ハイブリッドアプローチ

実用的な音声合成の観点からは、CFD（流れ計算）と音響伝播計算（FDTD、DWM、波動管）を**分離したハイブリッドアプローチ**が最も現実的である。

基本的な構成：

```
[近距離場：CFD または簡略流体モデル]
        ↓  音源項（体積速度 / 圧力変動 / 乱流強度）の抽出
[遠距離場：音響波動伝播モデル（DWM / FDTD / 波動管）]
```

代表的なハイブリッド実装：
- **Hybrid CFD + Lighthill アナロジー**（Springer Biomechanics 2015）：喉頭周辺を有限体積 CFD で解き、乱流音源項を抽出して声道以降を音響場で計算
- **Hybrid LES + FEM 音響**（Kniesburges ら, Acta Acustica 2024）：流体-構造-音響連成のベンチマークケース
- **2.5D FDTD**（Mohapatra ら, Interspeech 2024）：2D FDTD の速さを維持しつつ奥行き方向を加えた近似手法

このハイブリッド構成は、**本プロジェクトの DWM や波動管モデルに流体力学的音源を接続する際の直接的な枠組み**となる。

## 5. 風楽器・管楽器との比較

声帯と管楽器は「気流誘起振動 + 管共鳴」という共通の物理機構を持つ。この比較は発声モデルの設計に有益な示唆を与える。

### 5.1 フルート・リコーダーのエアリード音源

フルート類の音生成メカニズムは以下の流れで記述できる。

```
気流 → エッジ（labium）への衝突 → 渦放出 → 管内定在波との結合 → 正帰還ループ → 安定発振
```

Ségoufin ら（Meccanica）によれば、音響エネルギー生成はジェット下流域に集中し、渦放出が音響エネルギーの吸収と供給の両方に寄与するという双方向性がある。

P. de la Cuadra（CCRMA 博士論文, Stanford）による包括的な物理モデリングでは、jet-drive モデルとして以下のパラメータが重要とされる。

- **W/h 比**（エッジ幅と射出距離の比）
- **エッジ角度**（フルートと記録管で大きく異なる：60° vs 15°）
- **ジェット速度と音響速度の比**

これらは声帯–声道の場合の「声門幅・声門開度・閉鎖速度」に対応する概念として解釈できる。

### 5.2 クラリネット・リードの FSI

リードは気流誘起の自励振動素子であり、声帯との構造的類比が成立する。

最新の数値研究（ScienceDirect 2024）では、2D リード変形モデルと3D 圧縮性 Navier-Stokes を連成させた FSI が実装され、リード開口部でのジェット形成・流れ剥離・摩擦の非線形特性が再現されている。

クラリネットの**準定常流モデル**では、リード開口部での Bernoulli 型の圧力–流量関係に非線形性が現れ、これが自励振動の本質となる。声帯の two-mass モデルにおける非線形バネ特性と本質的に同じ数学的構造を持つ。

### 5.3 声帯と管楽器の比較表

| 特徴 | 声帯 | 管楽器リード | フルート/エアリード |
|---|---|---|---|
| 振動・変動の性質 | 開閉弁（自励振動） | 開閉弁（自励振動） | ジェット偏向（自励） |
| 流れとの連成 | 声門流–声帯FSI | 管内圧–リードFSI | ジェット–エッジ渦 |
| 主要音源種別 | 単極 + 双極 | 単極（主） | 双極 |
| 共鳴器 | 声道（断面積関数） | 楽器管（円柱/円錐） | 楽器管 |
| 主要フィードバック | 音響圧→声帯変位 | 音響圧→リード変位 | 音響圧→ジェット偏向 |
| 音高の制御 | 声帯張力・声門下圧 | 管長・リード剛性 | 管長・ジェット速度 |

管楽器の物理モデリング（特に JSIM, IRCAM PhysModel, Synthesis ToolKit STK など）で蓄積された知見は、発声モデルの設計に直接転用できる側面が多い。

## 6. 摩擦音・無声音の実用的モデル

### 6.1 VocalTractLab のエアロアコースティックモデル

Birkholz（2006）が VocalTractLab 向けに実装したモデルは、流体力学理論を実用的な音声合成に落とし込んだ代表例である。

音源配置の基本設計：

```
声道形状
    ↓
収縮点（最小断面積部位）の特定
    ↓
[単極ノイズ源] → 収縮部近傍に配置（振幅 ∝ U² × Re^0.5）
[双極ノイズ源] → 障害物位置（歯・口唇）に配置、2次 LPF で成形
```

双極源のカットオフ周波数は障害物のサイズと流速から計算され、4〜8 kHz 付近が典型値となる。

Stevens（1971）の理論的基盤によれば、乱流ノイズの振幅は

```
音源強度 ∝ ρ × U^n / A_constriction^m
```

の形をとり、速度指数 n は音源種別に応じて 4〜6 の範囲をとる。

### 6.2 声門吸気ノイズ（Aspiration noise）のモデル

声帯を通過する乱流吸気ノイズは以下の条件を基にモデル化できる。

- 声門体積速度 U が閾値 U_th（≈ 200〜400 cm³/s）を超えるとノイズが発生
- ノイズ振幅 ∝ (U - U_th)^2
- 声道の低域フォルマントで整形されて放射される
- LF モデルや Rosenberg モデルの周期成分に加算して合成波形を作る

このモデルは計算コストが非常に低く、波形バッファへの加算として実装できる。

## 7. 重要な知見の整理

### 7.1 音源種別と声道の分離（Source-Filter 理論の拡張）

古典的な Source-Filter 理論では、音源（声帯）と声道（フィルタ）を完全に独立して扱う。しかし流体力学的観点からは、この仮定が破れる重要なケースがある。

- **声帯–声道音響結合**：声道のインピーダンスが声帯振動に逆方向に影響する
- **声帯後方の乱流と声道共鳴**：声門ジェットの渦構造が声道内で整形される
- **Coanda 効果による非対称性**：ジェット偏向が声道内音場の非対称的な励振を生む

これらは特に**非常態的な発声**（囁き、ノイジーな声、モーダルと裏声の境界域）でより顕著に現れる。

### 7.2 音源の空間分布

発声の音源は一点に集中しているわけではなく、以下のように空間的に分布する。

| 音源位置 | 音源種別 | 対応する音声現象 |
|---|---|---|
| 声門（開閉タイミング） | 単極（体積速度変化） | 有声音の基本音源 |
| 声門後方（ジェット） | 四重極（乱流） | ブレシー声・声門吸気音 |
| 声道収縮部 | 単極（乱流） | 摩擦音（主収縮部） |
| 障害物近傍（歯・口唇） | 双極（衝突） | 歯擦音 /s/ /ʃ/ の主音源 |
| 口唇 | 双極（放射） | 放射による音響インピーダンス変化 |

この空間分布を波動管や DWM モデルの「途中に音源項を注入する」という設計に反映できる。

## 8. このプロジェクトへの応用可能性の整理

### 8.1 短期：パラメトリック・ノイズ源の導入（実装コスト低）

現在の DWM・波動管モデルに流体力学的知見を組み込む最も現実的な方法は、Birkholz モデルを参考にした**パラメトリック・ノイズ源**の追加である。

実装概要：
1. 声道断面積関数から収縮点（最小断面積部位）を特定する
2. 収縮点での Reynolds 数（Re）を推定する（Re = U × D / ν、D は水力直径）
3. Re > Re_th のとき、収縮点にフィルタ済みホワイトノイズを単極として注入する
4. 下流の障害物位置に双極ノイズ源を追加する（2次 LPF, 4〜8 kHz カットオフ）
5. 声門体積速度が閾値を超えたとき、声帯後方に吸気ノイズを加算する

このアプローチは：
- Python + NumPy のみで実装可能
- DWM や波動管モデルの現行実装を大きく変更しなくて良い
- 摩擦音（/s/, /ʃ/, /f/）・吸気音・ブレシー声の合成が初めて可能になる

### 8.2 中期：準1次元流れモデルとの連成（実装コスト中）

声道断面積関数から気流を解く**準1次元流れモデル（quasi-1D flow model）**を実装し、DWM の音源として利用する。

```
声道断面積関数 A(x)
    ↓
質量・運動量・エネルギー保存式（Bernoulli + 境界層損失）
    ↓
各断面での圧力 p(x)・速度 u(x)・Reynolds 数 Re(x)
    ↓
乱流遷移判定 → 音源強度・位置の推定
    ↓
DWM / 波動管モデルへの音源注入
```

この設計では、声道形状の変化に応じて音源の位置・種別・強度が動的に変化するため、より物理的に正確な子音遷移が表現できる。

追加として、Coanda 効果を模した**確率的なジェット偏向**（乱数で左右どちらかに偏る）を声門モデルに組み込むことで、音声の自然な非周期性（jitter / shimmer）を再現できる。

### 8.3 中長期：LBM による声門局所モデル（実装コスト高・研究価値高）

格子ボルツマン法（LBM）を用いた声門周辺の局所流れモデルを研究ワークスペースで実装する。

設計方針：
- 計算領域を声門付近（3〜5 mm 程度）に限定することで計算コストを抑える
- 声帯は rigid wall（剛体壁）と仮定し、振動波形を境界条件として与える（FSI は省略）
- LBM 格子を 0.1〜0.2 mm 分解能で設定（10〜20 セル × 30〜50 セル 程度）
- 出力は声門後方の圧力変動時系列として、波動管・DWM の入力端に渡す

この実装は全体 CFD（LES/DNS）より2〜3桁軽く、Python + NumPy で試作可能な水準に収めることが現実的に期待できる。

### 8.4 長期：完全流体-構造-音響連成（研究レベル）

FEM による声帯組織モデル + LES または LBM の流れモデル + DWM または FDTD の音響モデルの完全連成は、現状では研究機関レベルの計算資源を必要とし、リアルタイム利用は不可能である。

ただし、このプロジェクトの方針として**「精度優先・リアルタイム性不要」**が明確化されているため、長期的にはオフライン生成で完全連成モデルを検討する価値がある。
